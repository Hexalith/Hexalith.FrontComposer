## Problem

Duplicate Level 3 slot override.

## Common Causes

The SourceTools producer emits `HFC1040` when it detects `Duplicate Level 3 slot override` during FrontComposer validation or runtime reporting. Inspect the diagnostic payload for the affected symbol, generated path, package, or migration record before changing code.

## How To Fix

Follow the registry-owned message template for `HFC1040` and update the affected SourceTools surface. Keep evidence project-relative and sanitized, then rerun docs validation so the help link, lifecycle state, and related diagnostics stay synchronized.

## Example

```text
HFC1040: What: Duplicate Level 3 slot override. Expected: The SourceTools contract for HFC1040 is satisfied. Got: A bounded diagnostic payload identifies the affected FrontComposer surface. Fix: Apply the SourceTools remediation and rerun validation.
```

## Suppression Guidance

Suppression is `allowed-with-rationale`. Prefer `.editorconfig` or a narrowly scoped pragma for analyzer diagnostics.

## Migration/Deprecation

No migration action is required unless the registry later changes this diagnostic lifecycle.
