## Problem

Level 3 slot contract version is incompatible.

## Common Causes

The current Level 3 field-slot path reports `HFC1041` from `ProjectionSlotRegistry` at startup. Invalid or major-incompatible contract versions are ignored and generated default rendering wins. Minor drift is accepted and logged at Information level. It is not currently emitted as a SourceTools build diagnostic for compile-visible slot registrations.

## How To Fix

Rebuild the slot component against the installed FrontComposer package. If the major version differs, update the override package or disable the registration until it targets the current Level 3 slot contract.

## Example

```text
HFC1041: Level 3 slot descriptor has incompatible contract version. Expected: installed Level 3 slot contract version. Got: an older or newer incompatible major version. Fix: rebuild the slot component against the installed framework.
```

## Suppression Guidance

Suppression is `allowed-with-rationale`. Prefer `.editorconfig` or a narrowly scoped pragma for analyzer diagnostics.

## Migration/Deprecation

No migration action is required unless the registry later changes this diagnostic lifecycle.

None currently.
