## Problem

Custom override motion has no reduced-motion fallback.

## Common Causes

The SourceTools accessibility analyzer emits `HFC1054` at build time when statically inspectable custom override source contains animation, transition, or keyframes evidence without `prefers-reduced-motion` fallback evidence.

## How To Fix

Add a `prefers-reduced-motion` fallback or remove the motion from the override. Keep the fallback evidence in source inspected by the analyzer.

## Example

```text
HFC1054: What: A customization component introduces motion without reduced-motion evidence. Expected: Custom override motion respects reduced-motion preferences. Got: animation, transition, or keyframes evidence without prefers-reduced-motion. Fix: add a reduced-motion fallback or remove the motion.
```

## Suppression Guidance

Suppression is `allowed-with-rationale`. Prefer `.editorconfig` or a narrowly scoped pragma for analyzer diagnostics.

## Migration/Deprecation

No migration action is required unless the registry later changes this diagnostic lifecycle.

None currently.
