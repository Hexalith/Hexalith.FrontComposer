## Suppression Guidance

Suppression is `allowed-with-rationale`. Prefer `.editorconfig` or a narrowly scoped pragma for analyzer diagnostics.

## Migration/Deprecation

QueryRequest.Filter is replaced by ColumnFilters in v0.2.0; removed in v0.4.0. See HFC0001 for the deprecation contract and the [Obsolete] DiagnosticId / UrlFormat metadata.

Migration steps:
1. Replace `QueryRequest.Filter = ...` with `QueryRequest.ColumnFilters = ...`.
2. Update any custom serializers or projections that still read the legacy `Filter` property.
3. Rebuild against ≥ v0.2.0 and confirm no HFC0001 diagnostics remain in the build log.
