# FR24–FR29 Traceability Matrix (Story 5-7)

This matrix is the AC8 evidence surface for Story 5-7. Each row records:

* **FR**: the realtime resilience requirement.
* **Owner**: the production component or service that must satisfy the FR.
* **Harness scenario**: the harness staging primitive(s) that exercise the FR.
* **Default-lane test**: the deterministic xUnit/bUnit test class executing in the default CI lane.
* **Live smoke**: any non-default smoke test (rare; not counted toward the 90% denominator).

The denominator for AC8's "≥90% deterministic coverage" is the FR24–FR29 behavior rows in this
table — not global solution coverage. Adding a new fault selector must add or update a row here
before the selector counts toward AC8.
